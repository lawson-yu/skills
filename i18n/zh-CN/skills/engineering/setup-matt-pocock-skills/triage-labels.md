# 分诊标签

这些技能以五个规范的分诊角色来表述。此文件将这些角色映射到本仓库问题跟踪器中实际使用的标签字符串。

| Label in mattpocock/skills | Label in our tracker | 含义 |
| -------------------------- | -------------------- | ---------------------------------------- |
| `needs-triage`             | `needs-triage`       | 维护者需要评估此问题 |
| `needs-info`               | `needs-info`         | 正在等待报告者提供更多信息 |
| `ready-for-agent`          | `ready-for-agent`    | 规格完整，已可交给 AFK agent |
| `ready-for-human`          | `ready-for-human`    | 需要人工实现 |
| `wontfix`                  | `wontfix`            | 不会处理 |

当某个技能提到一个角色（例如“应用 AFK-ready 分诊标签”）时，请使用此表中对应的标签字符串。

编辑右侧列以匹配你实际使用的术语。
